# Exercícios em Java 

Repositório criado para armazenar exercícios básicos em Java,
com foco em lógica de programação e estruturas condicionais.

- Variáveis
- if / else
- Operadores lógicos
- Organização de código
- Entrada de dados com scanner
- Métodos e classes
